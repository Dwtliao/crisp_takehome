## üé• Recording Tips

### Before You Start:
- Clear browser's localStorage to start fresh
- Restart server for clean logs
- Close unnecessary browser tabs
- Test microphone for voice notes demo
- Check volume for TTS demo

### During Recording:
- Speak clearly and confidently
- Pause briefly between major sections
- Point to UI elements as you explain them
- Use phrases like "Notice..." and "Look at..." to direct attention

### Common Mistakes to Avoid:
- Don't rush - let the AI generate (it takes 3-5 seconds)
- Don't forget to show the warning message on Oceanique
- Don't skip the persistence demo - it's the most impressive part
- Don't forget to allow microphone permissions for voice notes
